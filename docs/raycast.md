# Raycast (macOS Spotlight Replacement)

## App Shortcuts

- Switch to Browser: `hyper + b`
- Switch to Code Editor: `hyper + c`
- Open File Explorer: `hyper + f`
- Switch to Obsidian Notes: `hyper + n`
- Switch to Music App: `hyper + m`
- Switch to Terminal: `hyper + t`

## Window Management

No window gap

- Left Half: `option + h`
- Bottom Half: `option + j`
- Top Half: `option + k`
- Right Half: `option + l`
- Maximize Window: `option + m`
- Toggle Fullscreen: `option + f`
- Center Window: `option + c`

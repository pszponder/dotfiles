# Custom NeoVim Config

## How to backup your current nvim files

```bash
# required
mv ~/.config/nvim{,.bak}

# optional but recommended
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}
```

## Entry Point Configuration File w/ init.lua

`init.lua` located in `.config/nvim` is the main entry-point for your NeoVim configuration.
